## 📖 简介

本专案提供了一套基于 **AutoHotkey v2** 编写的《QQ三国》自动化脚本。它利用底层的 Win32 API (`PostMessage`) 实现了物理输入与脚本输入的完美隔离，让您在多开游戏窗口或在后台挂机时，前台依旧可以毫无干扰地进行日常办公或其他操作。

本项目采用 **主控台 + 子进程** 的架构，通过一个统一的极简界面管理多个职业宏，彻底解决多开环境下的混乱问题。

## ✨ 核心特性

- **🛡️ 纯净安全**：基于 AutoHotkey 原生语法，无外部注入，仅模拟键鼠操作。
- **🎮 多角色适配**：针对游戏中不同职业的技能输出循环（Rotation）进行深度定制。
- **🔄 多开与后台**：利用 `PostMessage` 进行后台按键发送，支持多实例独立运行，不占用系统焦点。
- **⚡ 线程安全**：通过原子级的状态机与 `Critical` 临界区控制，确保复杂连招不会互相阻塞，并主动避让玩家物理操作。
- **🖥️ 统一控制台**：通过 `Main.ahk` 启动，提供选项卡式的界面（Tab），内置窗口自动检索与 PID 识别，不再需要繁琐的手动绑定。

## 👥 角色与职业支持

目前脚本库主要提供了以下职业的专属操作模块：

| 模块 (Tab) | 对应角色 | 职责 / 特点 |
| :--- | :--- | :--- |
| 🗡️ **`JS.ahk`** | **剑侍** | 包含自动连击、高爆发技能循环优化 (S→D→F→Q)，快速清理近身怪物。 |
| 🪄 **`XS.ahk`** | **仙术** | 包含定时释放的大招技能 (F、Q、W) 以及持续的长按 A 普攻循环。 |
| 🪭 **`YX.ahk`** | **仙术 (游侠)** | 综合了定时独立技能 (Q/W) 与循环技能 (S→D→F) 的混合输出流派。 |

> *注：上述模块均被内嵌在主控制台中，依赖底层的 `MacroCore.ahk` 提供核心的消息发送与状态调度能力。*

## 🚀 快速上手

### 环境要求
- **操作系统**：Windows 10 / 11
- **运行环境**：[AutoHotkey v2.0](https://www.autohotkey.com/) 及以上版本

### 使用步骤
1. **获取代码**：克隆或下载此代码仓库到您的本地目录。
2. **启动控制台**：双击运行 **`Main.ahk`**。（脚本会自动请求管理员权限以确保对游戏窗口的控制力）。
3. **选择职业与窗口**：
   - 在控制台中点击对应的职业标签卡（JS宏 / XS宏 / YX宏）。
   - 在下拉菜单中选择你要绑定的游戏窗口（下拉列表会自动显示所有窗口及对应的 `PID`，方便多开识别）。
4. **配置参数**：根据需要勾选对应的技能、设置循环次数或定时释放的时间（分钟）。
5. **启动挂机**：点击界面上的启动按钮或按下对应热键，脚本即可开始后台运转。

## ⌨️ 常用快捷键

| 快捷键 | 功能说明 |
| :--- | :--- |
| `1` / `2` / `3` | **[主界面内]** 快速切换对应的职业宏选项卡 (JS / XS / YX) |
| `Space` | **[可选]** 允许使用空格键单独暂停当前锁定的窗口宏 |

*(其他专属的启动/停止快捷键，请直接参考控制台界面内的按钮与提示)*

## 🛠️ 项目结构

```text
📁 AutoHotkey/
 ├── 📄 Main.ahk         # 🌟 主入口：启动多进程控制台并挂载子UI
 ├── 📄 MacroCore.ahk    # 核心底层：提供线程安全的键鼠操作隔离与状态机
 ├── 📄 JS.ahk           # 剑侍 逻辑控制子进程
 ├── 📄 XS.ahk           # 仙术 逻辑控制子进程
 ├── 📄 YX.ahk           # 游侠/仙术 逻辑控制子进程
 ├── 📄 MacroConfig.ini  # 配置文件（自动保存与加载各窗口的UI状态）
 └── 📄 README.md        # 项目说明文档
```
