# QQ三国 全能仿生宏引擎 (Sanguo Bio-Macro Engine)

基于 AutoHotkey v2.0 构建的专业级游戏辅助框架，专为《QQ三国》多开、后台挂机及高频操作设计。

## 🌟 核心架构与模块化

本项目经历了深度抽象化重构，核心驱动模块 (`MacroCore.ahk`) 已被精简为统一的入口文件，底层核心逻辑被完全解耦并拆分至 `Core/` 文件夹下，形成现代化的模块化架构：
- **`Core/InputManager.ahk`**: 底层输入与窗口管理器。完全负责目标窗口的探测过滤，以及通过 `PostMessage` 和 `ControlClick` 实现物理隔离级别的后台键鼠注入。
- **`Core/SyncManager.ahk`**: 同步操作控制器。独立管理所有的跨窗口键盘监听、鼠标轮询拦截，以及复杂的 Shift 翻转状态分发。
- **`Core/MacroEngine.ahk`**: 核心状态机引擎。专注负责自动化任务流水线（定时大招、循环技能、长按普攻），包含仿生时间抖动和避让机制。
- **`Core/MacroApp.ahk`**: 应用程序与调度层。作为极轻量级的调度枢纽，负责加载配置、实例化上述核心组件、注册外层热键响应。
- **`Core/MacroUI.ahk`**: 可视化 UI 抽象层。彻底消除了冗余的界面构建代码，通过组合式方法提供统一的自适应折叠面板与状态展示。
- **职业入口脚本 (JS / XS / YX / HL)**: 实际的使用入口，通过继承通用基类将千行的功能代码浓缩至不足 50 行，让开发者可以专心维护各职业的专属连招优先级。

## ⚡ 极限性能调优

为了保障多开场景下的系统资源稳定与按键极速响应，底层代码进行了极限优化：
- **1ms 高精度定时器**：利用 `DllCall("winmm\timeBeginPeriod", "UInt", 1)` 将系统默认的 15~16ms 调度精度强行提升至 1ms，极大减少了高频连按与技能穿插时的延迟漂移。
- **强制单例模式**：所有职业脚本强制引入 `#SingleInstance Force` 与严格的锁机制，杜绝因为误操作多开同一脚本导致的键盘钩子冲突与 CPU 占用飙升。
- **消息队列降压**：在 `PostMessage` 的极高频发包死循环中加入了极微小的 `DllCall("Sleep", 1)` 挂起，主动为系统底层的消息队列留出处理间隙，有效防止长按（Down/Up）事件在游戏引擎内乱序或颠倒。
- **最快事件投递**：全局兜底应用 `SendMode("Input")`，移除所有虚拟按键延迟 (`SetKeyDelay(-1)` 等)，确保任何非后台注入性质的辅助按键也能以最低延迟送达。

## 🛡️ 顶级防检测与仿生机制

- **Box-Muller 正态分布抖动 (`_Jitter`)**: 彻底抛弃虚假的平均随机器，将所有按键动作模拟成带有钟形曲线方差的人类按压时差。**所有按键间隔均存在 1.5x 的仿生随机波动**，波动范围安全合理。
- **拟人化反应硬直 (`_HumanStepDelay`)**: 默认开启紧凑型仿生操作时序，每个按键动作的执行前都加入了相当于普通熟练玩家 (50ms~80ms) 的反应延迟，告别机械脚本特征。
- **🌬️ A 键自动随机“换气”防检测**: 检测到过长的持续物理按压时（如几秒钟），随机产生数十毫秒级停顿以模拟真实手指换气。
- **☕ 疲劳系统（自动休息）**: 单独为持续挂机加入了疲劳休息轮询算法（每 2-4 分钟自动休息几秒），此系统**与定时技能系统独立分离**，休息时不影响定时大招的触发。

### 🔗 操作同步模式

多窗口同步操作功能，所有快捷键在**激活任一 QQ三国 窗口内**均可生效。

| 快捷键 | 功能 | 说明 |
| :--- | :--- | :--- |
| `数字键` (1/2/3/4) | 宏启停 | 启动/暂停对应职业的宏引擎 |
| `;` | 快捷键总开关 | 全局开启/关闭宏快捷键响应 |
| `RShift + 数字键` | 操作同步模式 | 开启后**键盘同步默认生效**，屏蔽干扰按键 (如数字区)，智能判定当前游戏窗口。 |
| `5` | 鼠标同步切换 | 仅在操作同步模式开启后可用，一键开启最高 60fps 的后台鼠标移动及完美双击同步。 |
| `LShift` (按住) | 临时翻转同步 | 开启同步时：按住暂停所有同步；关闭同步时：按住临时向全局发送键鼠同步。 |

### 🔔 唤灵职业宏详细介绍
*定位为法术型输出职业，以法铃为武器，兼具高频施法与强大的生存控制能力。*
- **核心机制**: 利用高频普攻和技能施放，通过核心技能获取可吸收伤害的“唤灵之盾”。
- **高阶技能托管**: 宏引擎自动调度高等级阵法与大招（如灵鲲霄浪等脱手群攻、云篆玄灵阵增益）。
- **挂机策略**: 引擎优先保障高等级大招（E/R/T）在冷却完毕后释放，并在空隙期间高频轮转基础循环技能（S/D/F/Q），最后以长按A键铃鸣普攻兜底，保障无缝连招输出。

## 🚀 快速开始

1. **环境准备**: 确保已安装 [AutoHotkey v2.0+](https://www.autohotkey.com/)。
2. **权限要求**: 若游戏以管理员权限运行，你需要右键以“管理员身份运行”宏脚本，否则系统会由于安全隔离 (UIPI) 导致宏失效。（脚本自带提权检测）
3. **启动使用**: 直接双击您对应职业的脚本 (例如 `JS.ahk`)，在控制面板中选择窗口，按下快捷键即可开始后台工作。
